# Koleksi Buku Novel

Aplikasi web untuk mengelola koleksi buku novel dengan fitur-fitur lengkap seperti manajemen buku, penulis, genre, dan sistem peminjaman.

## Fitur Utama

- Manajemen buku novel (tambah, edit, hapus, lihat detail)
- Manajemen penulis
- Manajemen genre
- Manajemen rak buku
- Sistem peminjaman dan pengembalian buku
- Dashboard untuk melihat ringkasan
- Sistem otentikasi pengguna

## Struktur Folder

Lihat dokumentasi struktur folder untuk penjelasan lebih lanjut tentang organisasi file dan direktori.

## Instalasi

1. Clone repository ini
2. Buat database MySQL sesuai dengan schema di `database/schema.sql`
3. Konfigurasi koneksi database di `config.php`
4. Jalankan aplikasi di server web (direkomendasikan XAMPP)
